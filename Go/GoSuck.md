## 1. Docstring

Most of the languages use `/*` for their docstring yet Go use `//`.

大部分的語言都使用`/*`當做`doc`, 不過Go反其道而行使用`//`. (可惡)

Why not this?

```go
/**
 * Description over here.
 * 
 * @param ... : ...
 * @return ...
 */
```

Use this?

```go
// Description over here. 
// @param ... : ...
// @return ...
```
